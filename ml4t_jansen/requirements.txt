### ⚠️ Critical Installation Notes

Since you are using `zipline-reloaded` and `TA-Lib` on Windows, standard `pip install` often fails because it cannot compile the C extensions.

If you ever need to recreate this environment, run these commands in this **exact order**:

1. **Create Environment:**
```bash
conda create -n ml4t python=3.10
conda activate ml4t

```


2. **Install Compiled C-Libraries via Conda (Crucial Step):**
* `numpy` and `ta-lib` must be installed from `conda-forge` *before* pip installing the rest to avoid compilation errors.


```bash
conda install -c conda-forge numpy ta-lib pandas

```


3. **Install the Rest:**
```bash
pip install -r requirements.txt

```
